# WinPE
De aquí podemos descargar la ISO del WinPE:
https://drive.google.com/drive/u/1/folders/1-UguLj4tkW2BcDIBZ29IrDS9DHK7s2_b

I. Preparación:

  1. Con algun software (Ventoy.net) creamos el medio de instalación con la ISO del WinPE
  2. Descargar la carpeta _Scripts_.
  3. La carpeta _Scripts_ la colocamos en la raiz del medio de instalación.

II. Instalación:

  1. Booteamos con el medio de instalación.
  2. Verificamos que el equipo esté correctamente conectado a internet.
  3. Particionamos y formateamos respondiendo las preguntas de disco duro a utilizar y Overprovisioning (opcional).
  4. Aguardamos que termine la instalación.
